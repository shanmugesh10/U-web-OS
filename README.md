# U-WebOS

## INTRODUCTION
- This is my first ever web related project.<br>
- Through this project I have learnt HTML, CSS & JAVASCRIPT.
- The umbrella os is an elegant, easy to use user-friendly os built by me from scratch without any ai help.<br>

## WHAT'S NEW (LATEST UPDATE)
- **Complete UI Overhaul:** Upgraded to a sleek, modern "Glassmorphism" design with a dark, moody Umbrella Corporation-inspired aesthetic.
- **Secure Lock Screen:** Added a functional boot-up lock screen with a dynamic clock. (Default PIN: `admin`).
- **Bio-Hazard Simulator (Live Widget):** A desktop widget featuring an auto-playing AI Snake game. Click the widget to take manual control!
- **Bio-Scan (Camera App):** A functional webcam viewer that allows users to snap photos and save them to a session gallery.
- **U-Net Explorer:** A fully functional in-OS web browser to surf the internet.
- **U Notes:** A notepad application featuring auto-save functionality using browser Local Storage.
- **U Calc:** A fully integrated desktop calculator.
- **Interactive Desktop:** Added draggable windows, z-index layering (bringing active windows to the front), and a dynamic wallpaper changer.
- **"Do Not Open" Protocol:** Added a hidden jumpscare app for an authentic horror-survival vibe.

## FEATURES
- Visually appealing and highly interactive
- User friendly drag-and-drop window management
- Real-time system clock and date processing
- Persistent data storage for user notes
- Hardware integration (Webcam access)
